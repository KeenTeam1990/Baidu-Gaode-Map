## 项目环境

项目使用 cocoapods 来管理第三方库，所以需要安装 cocoapods，安装方式网上有很多教程，这里不在详述。

- Xcode 8.2.1
- iOS 8.0

使用到的第三方库：

platform :ios, "8.0"

target "GaodeMapTest" do

pod 'BaiduMapKit'

end

platform :ios, "8.0"

target "baiduMapTest" do

pod 'Amap3DMap'

pod 'AmapSearch'

pod 'AmapLocation'

end
